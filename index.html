John Resig - Software Engineer and Entrepreneur

Creator of jQuery, a popular JavaScript Library
John Resig's history:
  • born August 8, 1984
  • 2004 - Worked with Ankur Teredesai on data mining instant messaging networks
  • 2005 - Graduated from Rochester Institute of Technology with a degree in Computer Science
  • 2006 - Created jQuery JavaScript Library
  • 2007 - Co-authored Apress book "Pro JavaScript Techniques"
  • 2012 - Authored "Secrets of the JavaScript ninja"

Resig has worked at Mozilla Corporation, and currently works for Khan Academy "... using JavaScript to teach JavaScript".
